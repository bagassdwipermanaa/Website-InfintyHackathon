## 🎯 Cara Menjalankan Aplikasi

### 1. **Development Mode (Recommended)**

```bash
# Windows PowerShell
.\scripts\dev.ps1

# Atau manual
npm run dev
```

### 2. **Manual Setup**

```bash
# Install dependencies
npm run install:all

# Start development servers
npm run dev
```

## 🌐 Akses Aplikasi

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **Health Check**: http://localhost:5000/health

